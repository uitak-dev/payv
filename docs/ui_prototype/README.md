# Lo-fi HTML Prototype (Tailwind + vanilla JS)

- Each page is a standalone HTML file.
- Tailwind is loaded via the official Play CDN script tag (prototype/development usage).
- Common assets:
  - assets/styles.css : layout helpers (centered container, fixed bottom nav/FAB)
  - assets/app.js     : simple interactions (nav active, accordion, tag max 3, attachments max 2 + preview)

Start:
- Open login.html or index.html in a browser.
